# Part 3: Menu, Click Dispatch, and Main-Goroutine Integration

The `tray.Run` function (called on the main goroutine) blocks on `systray.Run` until `systray.Quit`. It builds the menu, spawns the click-dispatch and refresh goroutines, and waits. This is the part that finally puts an icon in the menu bar.

### Task 8: `tray.Run` — main-goroutine entry, menu tree, dispatch loop, refresh loop

`tray.Run(ctrl, icon)` is the function the main goroutine calls. It calls `systray.Run(onReady, onExit)`, which blocks. `onReady` builds the menu and spawns two goroutines (dispatch, refresh). `Quit()` from `performShutdown` (in `serve.go`) unblocks `systray.Run` and the function returns.

**Files:**
- Create: `htrcli/internal/tray/tray.go` (the main entry point)
- Create: `htrcli/internal/tray/tray_disabled.go` (no-op stub for unsupported GOOS)
- Create: `htrcli/internal/tray/tray_test.go` (uses fakeController and fakeSystray under `traytest` tag)

**Interfaces:**
- Produces: `func Run(ctrl Controller, icon []byte)` — blocks until `Quit()` is called. On unsupported GOOS, the disabled-stub `Run` returns immediately.

- [ ] **Step 1: Stub `tray.go` with the public function signature**

In `htrcli/internal/tray/tray.go`:

```go
package tray

import (
    "log"
    "time"

    "github.com/getlantern/systray"
)

func Run(ctrl Controller, icon []byte) {
    systray.Run(func() { onReady(ctrl) }, onExit)
    // systray.Run blocks here until systray.Quit() is called.
}

func onReady(ctrl Controller) {
    systray.SetIcon(iconFromBytes(icon))
    systray.SetTitle("htrcli")
    systray.SetTooltip("htrcli daemon")

    // Build the menu. Item references are stored in package-level vars
    // so the dispatch and refresh goroutines can update titles.
    mStatus = systray.AddMenuItem("Status: …", false)
    mStatus.Disable()
    mLastErr = systray.AddMenuItem("Last error: —", false)
    mLastErr.Disable()
    systray.AddSeparator()
    mMaintenance = systray.AddSubMenuItem("Maintenance", "Maintenance actions")
    mReinstallChrome = mMaintenance.AddSubMenuItem("Reinstall (Chrome)", "Re-register the native host in Chrome")
    mReinstallFirefox = mMaintenance.AddSubMenuItem("Reinstall (Firefox)", "Re-register the native host in Firefox")
    mOpenConfig = mMaintenance.AddMenuItem("Open config folder", "")
    mCopyToken = mMaintenance.AddMenuItem("Copy bearer token", "")
    mShowLog = mMaintenance.AddMenuItem("Show recent log", "")
    mMaintenance.AddSeparator()
    mRestart = mMaintenance.AddMenuItem("Restart", "Restart the daemon")
    mQuit = mMaintenance.AddMenuItem("Quit", "Quit the daemon")

    // Spawn dispatch and refresh goroutines.
    ctx, cancel := context.WithCancel(context.Background())
    trayCancel = cancel
    go dispatchLoop(ctx, ctrl)
    go refreshLoop(ctx, ctrl)
}

func onExit() {
    if trayCancel != nil { trayCancel() }
}
```

(You will need the `iconFromBytes` helper, the `trayCancel` package var, and the `m*` MenuItem package vars. Add them in the same file.)

- [ ] **Step 2: Implement the dispatch loop**

In `tray.go`:

```go
func dispatchLoop(ctx context.Context, ctrl Controller) {
    defer func() {
        if r := recover(); r != nil {
            log.Printf("[htrcli tray] recovered from panic: %v", r)
        }
    }()
    for {
        select {
        case <-ctx.Done():
            return
        case <-mReinstallChrome.ClickedCh:
            runAction("Reinstall Chrome", func() error { return ctrl.ReinstallHost("chrome") })
        case <-mReinstallFirefox.ClickedCh:
            runAction("Reinstall Firefox", func() error { return ctrl.ReinstallHost("firefox") })
        case <-mOpenConfig.ClickedCh:
            runAction("Open config", ctrl.OpenConfigFolder)
        case <-mCopyToken.ClickedCh:
            runActionWithToast("Copy", ctrl.CopyTokenToClipboard)
        case <-mShowLog.ClickedCh:
            runAction("Show log", ctrl.OpenLog)
        case <-mRestart.ClickedCh:
            runAction("Restart", ctrl.Restart)
        case <-mQuit.ClickedCh:
            runAction("Quit", ctrl.Quit)
        }
    }
}

func runAction(name string, fn func() error) {
    if err := fn(); err != nil {
        flashError(name, err)
    }
}

func runActionWithToast(name string, fn func() (string, error)) {
    tok, err := fn()
    if err != nil {
        flashError(name, err)
        return
    }
    flashToast("Copied (" + fingerprint(tok) + ")")
}
```

- [ ] **Step 3: Implement the refresh loop and `refresh()`**

```go
func refreshLoop(ctx context.Context, ctrl Controller) {
    rate := refreshInterval()
    ticker := time.NewTicker(rate)
    defer ticker.Stop()
    refresh(ctrl) // initial paint
    for {
        select {
        case <-ctx.Done():
            return
        case <-ticker.C:
            refresh(ctrl)
        }
    }
}

func refresh(ctrl Controller) {
    s := ctrl.Status()
    mStatus.SetTitle(fmt.Sprintf("Status: %d · %d relays · %s", s.Port, s.RelaysConnected, okLabel(s.LastError)))
    if s.LastError == "" {
        mLastErr.SetTitle("Last error: —")
    } else {
        mLastErr.SetTitle("✗ Last error: " + truncate(s.LastError, 60))
    }
}
```

`refreshInterval()` returns `5 * time.Second` in production and `50 * time.Millisecond` under the `traytest` build tag (where `fakeSystray.TickRate()` provides the value).

- [ ] **Step 4: Implement `flashError` and `flashToast`**

Both write to a hidden status item (added in `onReady` but not visible to the user) for a duration, then restore. The 6s error / 2s toast durations match the spec.

```go
func flashError(name string, err error) {
    mFlash.SetTitle("✗ " + name + ": " + truncate(err.Error(), 60))
    go func() {
        time.Sleep(6 * time.Second)
        mFlash.SetTitle("")
    }()
}

func flashToast(msg string) {
    mFlash.SetTitle(msg)
    go func() {
        time.Sleep(2 * time.Second)
        mFlash.SetTitle("")
    }()
}

func truncate(s string, n int) string {
    if len(s) <= n { return s }
    return s[:n-1] + "…"
}
```

### Task 9: Wire `tray.Run` into `serve.go` (inverted main, performShutdown, signal handler)

This is the architectural change: the main goroutine now drives the tray; the HTTP server, signal handler, sweeper, and refresh loop run as background goroutines. Shutdown is explicit, not deferred.

**Files:**
- Modify: `htrcli/internal/commands/serve.go` (rewrite the main flow inside `serveCmd.RunE`)

**Interfaces:**
- Produces: `func performShutdown()` — the explicit shutdown sequence. `htrcli serve` no longer uses a single `defer d.Stop()` for daemon teardown; `d.Stop()` is called from `performShutdown`. The HTTP listener and Unix-socket listener are tracked at the `serveCmd.RunE` scope so they can be closed during shutdown.

sigCh := make(chan os.Signal, 1)
signal.Notify(sigCh, syscall.SIGINT, syscall.SIGTERM)

- [ ] **Step 4: Add `performShutdown` (stub for now; fleshed out in Part 4)**

```go
func performShutdown(trayAttached bool, srv *http.Server, d *host.Daemon, httpLn, unixLn net.Listener) {
    // 1. Quit the tray (idempotent if not started).
    if trayAttached {
        systray.Quit()  // (requires importing getlantern/systray; the spec accepts this import in serve.go IF needed for the Quit call. Alternative: expose tray.Quit() in the tray package.)
    }
    // 2. Drain HTTP (5s timeout).
    shutdownCtx, cancel := context.WithTimeout(context.Background(), 5*time.Second)
    defer cancel()
    if err := srv.Shutdown(shutdownCtx); err != nil {
        log.Printf("[htrcli serve] HTTP shutdown: %v", err)
    }
    // 3. Stop the daemon.
    d.Stop()
    // 4. Close the Unix-socket listener.
    if unixLn != nil { unixLn.Close() }
}
```

(To avoid importing `getlantern/systray` in `serve.go`, add a thin wrapper `func tray.Quit()` in Part 4. For now, the import is acceptable and can be removed later.)
